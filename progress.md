# MAIPS Project Progress

## Project Title
Multilayer Adaptive Intelligent Phishing System (MAIPS)

---

## Current Progress

# Layer 1 — URL Analysis

Implemented URL-based phishing detection using:

- URL length analysis
- IP address detection
- HTTPS verification
- Suspicious keyword detection
- Entropy calculation
- Brand spoof detection
- Subdomain depth analysis
- Double slash detection
- Special character checks

### Purpose

Detect suspicious patterns directly from the URL before opening the page.

---

# Layer 2 — ML Analysis

Implemented machine learning based phishing prediction using:

- Logistic Regression model
- StandardScaler
- Large phishing dataset training
- Confidence score generation
- Feature contribution analysis
- Risk score calculation
- Explainable output for suspicious factors

### Purpose

Predict phishing probability intelligently using trained ML model and weighted feature analysis.

---

# Layer 3 — Content Analysis + Intent Reasoning

Implemented content-based phishing detection using:

- Password field detection
- Form detection
- External form action detection
- Redirect script detection
- Suspicious word detection
- Known brand detection
- Threat type reasoning
- Severity classification
- Explainable AI output ("Why Flagged")

### Purpose

Understand attacker intent and explain why the website is suspicious instead of only giving phishing/not phishing result.

---

# Chrome Extension

Implemented browser extension with:

- Real-time current tab scanning
- Layer 1 + Layer 2 + Layer 3 result display
- Feature contribution graphs
- Content analysis graphs
- Professional dashboard UI
- Threat explanation panel
- Risk score visualization
- Severity classification display

### Purpose

Provide real-time phishing detection directly inside browser while users browse websites.
